Board recovery using the ISP pin
If the board's FLASH can no longer be programmed, the image in FLASH may be corrupt. To recover the board, the board must be reset with the ISP pin. To do this, hold down the SW6 button while pressing and releasing the reset button (SW1). On debuggers that reset the board prior to programming, the SW6 button should be held down until the programming cycle is started. Once started, it can be released.
